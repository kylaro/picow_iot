# Pico W IoT project

## How to use: 
Attach SCD41 sensor + LEDs to Pico W
Install MicroPython on Pico W
Use Thonny IDE to upload python files to Pico W
Set up backend in AWS to write / read DynamoDB
Upload website as static site to AWS S3
